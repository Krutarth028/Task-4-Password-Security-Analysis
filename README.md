# Task-4-Password-Security-Analysis
cybersecurity internship task-4

## Objective
To understand password storage, attacks, and authentication mechanisms.

## Tools Used
- John the Ripper
- Online Hash Cracking Tools
- Hash Generators

## Work Done
- Studied hashing vs encryption
- Analyzed common hash types
- Attempted password cracking on weak hashes
- Studied dictionary and brute-force attacks
- Learned importance of Multi-Factor Authentication

## Result
Weak passwords and outdated hashing algorithms were found to be insecure.

## Conclusion
Strong passwords, secure hashing algorithms, and MFA are essential for modern cybersecurity.
